# APP_NAME Features

This application is intended to be a go-to reference app that includes common components. The components are:
* Next.js front end with App Router
* .Net C# API backend
* Entity framework for API to query database
* PostgreSQL Database
* JWT authenticated interaction between the Next.js front end and .Net backend
* Responsive UI that is highly capable on mobile devices, using Tailwind CSS
* Docker image for running backend
* Technical SEO support including sitemap, robots.txt, metadata, and social previews
* Make files for:
  * Running .Net backend API in Docker
  * Running Postgres DB via Docker
  * Running Next.js application in Docker with a standalone Node runtime
* Landing page for users that displays some introduction text
